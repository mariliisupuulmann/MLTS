# MLTS – Mari-Liisu Training System

MLTS on treeningkavade koostamise süsteem, mis ühendab Mari-Liisu treenerimetoodika, teaduspõhise treeningprogrammeerimise ja AI otsustusmootori.

## Repo struktuur

- `docs/` – treenerifilosoofia ja dokumentatsioon
- `specs/` – tehnilised spetsifikatsioonid
- `data/` – harjutuste, lihaste ja reeglite andmemudelid
- `prompts/` – AI promptid ja süsteemijuhised
