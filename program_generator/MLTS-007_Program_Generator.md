# MLTS-007 Program Generator

## Eesmärk
Program Generator koostab kliendi sisendandmete põhjal tervikliku treeningkava.

## Sisend
- Client Profile Engine
- Decision Engine
- Exercise Intelligence
- Progression Engine
- Validation Engine

## Väljund
- Treeningjaotus
- Harjutused
- Seeriad
- Kordused
- Tempo
- Puhkepausid
- Alternatiivid
- Märkused

## Töövoog
1. Kontrolli kliendi profiili.
2. Vali sobiv treeningjaotus.
3. Vali põhiharjutused.
4. Lisa abiharjutused.
5. Arvuta maht.
6. Rakenda progressiooniloogika.
7. Käivita valideerimine.
8. Genereeri lõplik treeningkava.
